# Security Policy

## Supported Versions

Only the latest release receives security fixes.

## Reporting a Vulnerability

Report security issues privately via GitHub: the Security tab of
[github.com/genged/capshelf](https://github.com/genged/capshelf) →
"Report a vulnerability". Please do not open public issues for security
reports.

For capshelf's trust model and guidance for teams adopting shared
capabilities, see [`docs/security.md`](docs/security.md).
